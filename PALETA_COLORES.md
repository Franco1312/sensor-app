# Paleta de Colores - Econnexus

## Colores Principales

### Color Primario (Brand Color)
- **Amarillo/Dorado Principal**: `#F0B90B`
  - RGB: `rgb(240, 185, 11)`
  - Es el color de marca principal, inspirado en Binance
  - Usado para elementos destacados, botones principales y acentos

### Variaciones del Color Primario
- **Amarillo Oscuro**: `#D4A008` (primaryDark)
- **Amarillo Claro**: `#F5D047` (primaryLight)
- **Amarillo Hover**: `#F8D96A` (primaryHover)
- **Amarillo Presionado**: `#E5A807` (primaryPressed)

### Color de Splash Screen
- **Amarillo Suave**: `#F4D35E`
  - RGB: `rgb(244, 211, 94)`
  - Usado en pantalla de inicio y iconos adaptativos

## Colores de Estado

### Éxito/Positivo
- **Verde Principal**: `#0ECB81`
  - RGB: `rgb(14, 203, 129)`
  - Usado para indicadores positivos, precios al alza

### Error/Negativo
- **Rojo Principal**: `#F6465D`
  - RGB: `rgb(246, 70, 93)`
  - Usado para errores, precios a la baja

### Información
- **Azul**: `#1890FF`
  - RGB: `rgb(24, 144, 255)`

## Colores Neutros

### Grises Principales
- **Gris Oscuro (Texto)**: `#181A20`
- **Gris Medio**: `#5E6673`
- **Gris Claro**: `#848E9C`
- **Gris Muy Claro**: `#B7BDC6`

## Recomendación para Logo

**Colores sugeridos para el logo:**
1. **Color principal**: `#F0B90B` (Amarillo/Dorado)
2. **Color secundario**: `#181A20` (Negro/Gris oscuro) o `#0F0F0F` (Negro mate)
3. **Acento opcional**: `#0ECB81` (Verde) para elementos de crecimiento/prosperidad

**Combinaciones recomendadas:**
- Logo monocromático: `#F0B90B` sobre fondo oscuro o claro
- Logo con dos colores: `#F0B90B` + `#181A20`
- Logo con acento: `#F0B90B` + `#181A20` + `#0ECB81` (para elementos específicos)
